### typora 主题css 加载步骤

Typora将按以下顺序加载CSS文件：

1. Typora的基本款式。
2. 当前主题的CSS。
3. 主题文件夹下`base.user.css` 应用于所有主题。
4. 在主题文件夹下`{current-theme}.user.css`。如果您选择`Github`作为主题，那么`github.user.css`也将加载，应用于对应主题 。

如果它们不存在，您可以在主题文件夹下创建`base.user.css`和`{current-theme}.user.css`。

##### 搜索功能

打开高级设置，打开 `conf.user.json`文件，修改`searchService`

在视图/搜索打开搜索框（`ctrl+shift+f`）

```json
// 源文件  
// Array - Search Service user can access from context menu after a range of text is selected. Each item is formatted as [caption, url]

        ["Search with Google", "https://google.com/search?q=%s"]
    ],

//  修改后的文件
    "searchService": [
        ["Search with Google", "https://google.com/search?q=%s"],
        ["Translate", "http://translate.google.com/?source=osdd#auto|auto|%s"]

    ]
```
